# SHO-KEN コーポレートサイト

株式会社SHO-KEN（外壁塗装・リフォーム・便利屋）の公式コーポレートサイトです。

## 公開URL

https://CrowZz.github.io/sho-ken/

## 技術スタック

- HTML5 / CSS3 (Vanilla)
- JavaScript (ES2022, No framework)
- Google Fonts (Noto Sans JP, Montserrat)
- GitHub Pages でホスティング

## ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/CrowZz/sho-ken.git

# ブラウザで index.html を開く
# または Live Server 等を使用
```
